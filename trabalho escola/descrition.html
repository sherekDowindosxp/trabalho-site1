Descrição do Curso de HTML e CSS

Visão Geral

O curso de HTML e CSS é voltado para iniciantes que desejam aprender a criar páginas web utilizando essas linguagens fundamentais.

O que você vai aprender

  • Introdução ao HTML e CSS
  • Estrutura básica de uma página HTML
  • Formatação de textos e estilos com CSS
  • Inclusão de imagens e links
  • Criação de layouts responsivos
  • Utilização de frameworks CSS populares

Público-Alvo

Este curso é recomendado para estudantes, desenvolvedores iniciantes e qualquer pessoa interessada em aprender a criar websites.

Duração

O curso possui duração total de 1 ano e meio, com aulas semanais de 2 horas cada.

Certificado

Ao concluir o curso com sucesso, você receberá um certificado de conclusão reconhecido pelo mercado.

Investimento

O investimento para o curso de HTML e CSS é de R$ 600,00.
